# 概念

# 什么是脚手架？

一类方便快速形成工程化目录的工具，如：yeoman，@vue/cli，create-react-app...

# 什么是CLI？

CLI（Command-Line Interface,缩写：CLI）,如webpack，eslint，json-server...

# CLI工具的原理

  • commander 获取用户传参
  • inquirer 与用户进行交互，获取用户输入（选项）
  • bin 执行文件，cli 命令的入口
